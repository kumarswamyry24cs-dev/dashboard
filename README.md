# Smart City Emergency Recovery System

A dark-themed, futuristic mission control dashboard built with React + Tailwind CSS. Features real-time system monitoring, incident tracking, and emergency response coordination.

## 🎨 Design Features

- **Dark Cyberpunk Aesthetic**: Deep navy (#0a0f1e) background with neon cyan and green accents
- **Mission Control UI**: Glowing borders, scanline effects, and grid overlays
- **Professional Fonts**: Orbitron for headings, Share Tech Mono for data displays
- **Real-time Animations**: Pulsing glows, radar sweeps, and dynamic status indicators
- **Responsive Design**: Works seamlessly on desktop and tablet devices

## 🎯 Key Features

1. **System Status Dashboard**
   - Power Grid monitoring
   - Water Supply tracking
   - Communications Network status
   - Emergency Services coordination
   - Transport Network overview
   - Healthcare System monitoring

2. **City Map View**
   - 16-sector city grid visualization
   - Real-time incident mapping
   - Population impact tracking
   - Emergency controls and protocols

3. **Incident Log**
   - Real-time incident tracking
   - Severity-based color coding
   - Timeline visualization
   - Live incident updates

4. **Quick Statistics**
   - System uptime metrics
   - Average response times
   - Active incident count
   - Network status overview

## 🚀 Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Starts the development server at `http://localhost:5173`

### Build

```bash
npm run build
```
